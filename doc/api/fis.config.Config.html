set(path, value)

通过路径设置配置项。

注意：当设置路径时，如果目标路径已经存在，则会被覆盖。如果是多级路径，同时上级目录存在且值不为对象，则会报错。

如：

fis.config.set('a', 'sting');

// 如果这样设置会报错。
fis.config.set('a.subpath', 2);
