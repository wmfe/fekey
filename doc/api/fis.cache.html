fis. cache

用来创建 ~Cache 对象， 更多细节请查看 ~Cache 说明。

Source:
  • cache.js, line 127
See:
  • Cache 类说明

Example

// 检查该文件是否启用缓存
if (file.useCache) {

  // 如果启用，则创建 cache 对象
  file.cache = fis.cache(file.realpath, CACHE_DIR);

  // 保存内容以及信息到缓存中。
  file.cache.save('body', {
    foo: 1
  });
}
