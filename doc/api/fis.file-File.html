defineLikes() → {Undefined}

定义默认的文件类型，如：isHtmlLike、isJsLike、isCssLike。他们是互斥的，当设置其中某个值为 true 时， 其他属性应该为 false.

提取到一个方法里面原因是：isXXXLike 的属性是通过 Object.defineProperties 定义的，当 file 对象从文件 cache 里面反序列回来后，isXxxLike 系列的属性都会丢失。 所以这块需要重新调用（定义）一次。
